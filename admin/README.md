# FarmForce AI — Admin Dashboard

React-based admin dashboard for managing the FarmForce AI platform.

## Setup

```bash
cd admin
npm install
npm run dev
```

## Features

- 📊 **Dashboard**: Real-time stats (users, jobs, GMV, revenue) + charts
- 👥 **Users**: Manage farmers, workers, contractors with search and filters
- 💼 **Jobs**: View all job postings with status management
- 💳 **Transactions**: Monitor payments and escrow status
- ⚖️ **Disputes**: Handle and resolve user disputes

## Tech Stack

- **React 18** + **React Router v6**
- **Recharts** for data visualization
- **Tailwind CSS** for styling
- **Axios** for API calls
- **@heroicons/react** for icons
